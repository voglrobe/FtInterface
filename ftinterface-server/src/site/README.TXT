README.TXT

To run this tool the environment variable 'JAVA_HOME' must point to a
Java Runtime Environment (JRE or JDK) >= 1.7. Check with:

Linux:
   $JAVA_HOME/bin/java -version
   
Windows:
   %JAVA_HOME%\bin\java -version

Check 'Application.properties' to configure the database connections.
   
For a short description and a list of supported
command line options run the tool from within this directory:

Linux:
   ./startserver.sh ?

Windows:
   startserver ?
